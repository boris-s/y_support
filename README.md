# YSupport

Common support library for Y* gems.

## Installation

Add this line to your application's Gemfile:

    gem 'y_support'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install y_support

## Usage

Require 'y_support/all', or require 'y_support/something', and use it.
